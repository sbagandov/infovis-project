Project Info

During a time when everyone is quarantining at home, we decided to spend our time at home creating visualizations to look at how the number of coronavirus cases per state have affected the Google search queries for that state when compared to one year ago.

Some patterns were expected, but others were very surprising. We present our visualizations to you so that you can easily analyze the data and create your on conclusions.

The tree map gives us a different visualization to use to look for the correlation between a Google search query for 'coronavirus' in a state and the number of confirmed coronavirus cases for that state.

The bigger the square is, the more that state was searching for 'coronavirus' on Google. A square becomes darker shades of blue as the number of confirmed cases for that state increases.
